WORDLE ENGINE - Olivier Brossard-Roy

1 - REQUIREMENTS:

  rustup -> https://rust-lang.org/tools/install/

2 - TERMINAL COMMANDS:

  compile and run -> cargo run --release

3 - GAME:

  0 -> Automatic test of average score for all words
  
  1 -> Manual play

 4 - MAMUAL PLAY:

   1. Open wordle game in browser.
   2. Copy terminal-given word in the game.
   3. Copy output color-pattern in terminal
      
      Red (grey) = 'R'
      
      Green      = 'G
      
      Yellow     = 'Y'


  Example: "RRYGR"

  4. Repeat until end of the game.

  
That's it.
